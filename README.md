# 🥖 Bakery Website

A fully responsive bakery website built using **HTML5, CSS3, Bootstrap 5, and JavaScript**. The project focuses on creating a modern, clean, and user-friendly experience while following responsive web design best practices.


## ✨ Features

- Fully Responsive Design
- Modern Landing Page
- Responsive Navigation Bar
- Mobile Navigation Menu
- Sticky Navbar
- Smooth Hover Effects
- Image Gallery
- Contact Section
- Google Maps Integration
- Dropdown Navigation
- Cross-browser Compatible
- SEO-friendly Structure
- Clean and Organized Code
